Privacy Policy

ChatGPT Jump Panel

Overview

ChatGPT Jump Panel is a browser extension designed to improve navigation within ChatGPT conversations. Your privacy is important to us.

Data Collection

This extension does not collect, store, or transmit any personal data or user content.

Data Usage

The extension operates entirely locally in your browser. Any settings (such as panel position, colors, or preferences) are stored locally using the browser’s chrome.storage API.

Third-Party Services

This extension does not use any third-party analytics, tracking, or external services.

Network Requests

The extension does not make any network requests.

Changes to This Policy

If this privacy policy changes in the future, updates will be reflected on this page.

Contact

If you have any questions about this privacy policy, please open an issue on the project’s GitHub repository.
